# The House of Manna Website

Public website for The House of Manna, a Baltimore-area nonprofit combining food security, youth education, and community empowerment.

## Website

Once GitHub Pages is enabled from the `main` branch and repository root, the site will be available at:

`https://chadelliot.github.io/TheHouseofManna/`

## Deployment

- Homepage: `index.html`
- Hosting: GitHub Pages
- Publishing source: `main` branch, repository root
- `.nojekyll` is included so GitHub serves the repository as a standard static site.

## Form note

The “Stay informed” form currently displays an on-page confirmation but does not yet store submissions. A form backend such as Formspree, Mailchimp, HubSpot, or another service must be connected before collecting email addresses.
